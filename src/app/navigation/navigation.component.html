Graph Visualizer

This project aims to help you to create beautiful graphs only makink clicks. As you can see, you can create, update or deletes nodes and edges


How to draw?

For a Node

To create a node press SHIFT + Click

To change the position of a node maintain Click in a node

To delete a node press CRTL + Click

For an Edge

To create an edge, press ALT + Click on a node and then press ALT + Click in another node

To update an edge, press Double Click on an edge, then in the input, provide the new value for the weight
